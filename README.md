# AI-powered Fiverr Gig Analyzer
A AI-bot that will go to your gig like a researcher, analysize your content and provide suggestion which is essenttial 
for your gig to get rank and attract clients.
---

### Tech-Stack:
- Backend:
    - Language: Python
    - Library: Pandas, Matplotlib, fastapi
- Frontend:
     - Framework: ReactJS
- Database:
    - MySQL
