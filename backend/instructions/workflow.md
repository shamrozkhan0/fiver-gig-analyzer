# GigBro Workflow

Follow this workflow for every evaluation. Complete each step before moving to the next.

---

## Step 1 — Validate Input

Validate the provided gig information and identify:

* Missing fields
* Invalid values
* Inconsistencies

Continue using all valid information.

---

## Step 2 — Identify the Seller's Core Service

Determine:

* Primary service
* Supporting services
* Target audience

Use this understanding throughout the evaluation.

---

## Step 3 — Identify Keywords

Determine:

* Primary keyword
* Secondary keywords
* Supporting keywords

Ensure they accurately represent the seller's service.

---

## Step 4 — Determine Buyer Search Intent

Identify the buyer's primary goal and evaluate whether the gig satisfies that intent.

---

## Step 5 — Evaluate Components

Evaluate each component independently:

1. Category & Subcategory
2. Gig Title
3. Gig Description
4. Tags
5. Seller Profile
6. Packages & Pricing

Record strengths, weaknesses, and opportunities for improvement.

---

## Step 6 — Calculate Scores

Calculate:

* SEO Strength
* Keyword Strength
* Buyer Trust
* Package Review
* Pricing Review

Use the defined scoring rubric.

---

## Step 7 — Optimize Content

Generate optimized versions of:

* Gig Title
* Gig Description
* Tags
* Seller Profile Description

Improve existing content whenever possible.

---

## Step 8 — Final Verification

Verify that:

* Scores are internally consistent.
* Recommendations match the analysis.
* Optimized content aligns with the seller's service.
* Category recommendations are supported by evidence.
* The output satisfies the required schema.

---

## Step 9 — Return Result

Return the final response using the required JSON structure.
