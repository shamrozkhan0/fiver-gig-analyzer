# GigBro Scoring Rubric

## General Rules
Every score must be objective, evidence-based, and calculated only from the information provided.
Do not inflate or reduce scores because a gig has many orders, many reviews, or high ratings.
Scores should reflect SEO quality—not marketplace success.
All scores must be integers between **0 and 100**.

---
# SEO Strength (0–100)

SEO Strength measures how well the gig is optimized for Fiverr search visibility and buyer relevance.
Calculate the score using the following weighted components:

| Component                        |  Weight |
| -------------------------------- | ------: |
| Title                            |      25 |
| Description                      |      25 |
| Tags                             |      15 |
| Category & Subcategory Alignment |      10 |
| Seller Profile                   |      10 |
| Packages & Pricing               |      10 |
| Keyword Consistency              |       5 |
| **Total**                        | **100** |

---
## Title (25 Points)

Evaluate:
* Primary keyword relevance
* Primary keyword placement
* Search intent alignment
* Readability
* Natural wording
* Grammar
* Clarity
* Buyer appeal

Deduct points for:
* Keyword stuffing
* Generic wording
* Grammar mistakes
* Misleading wording
* Unclear service

---
## Description (25 Points)
Evaluate:
* Service clarity
* Readability
* Logical structure
* Keyword integration
* Search intent alignment
* Buyer-focused writing
* Professional tone

Deduct points for:
* Poor grammar
* Keyword stuffing
* Repetition
* Missing service details
* Poor readability

---

## Tags (15 Points)
Evaluate:
* Relevance
* Search intent
* Keyword diversity
* Coverage of the service
* Absence of duplicate intent

Deduct points for:
* Irrelevant tags
* Duplicate tags
* Overly broad tags
* Tags unrelated to the service

---
## Category & Subcategory (10 Points)
Evaluate whether:
* The selected category accurately matches the offered service.
* The selected subcategory is the best fit.
* The title, description, expertise, and tags all support the selected category.

Full score requires complete alignment.

---
## Seller Profile (10 Points)
Evaluate:
* Professionalism
* Clarity
* Consistency with the gig
* Relevant keyword usage
* Buyer confidence

Do not reduce this score because of ratings or reviews.

---
## Packages & Pricing (10 Points)
Evaluate:
* Value offered
* Feature completeness
* Clear differentiation
* Pricing fairness
* Logical package progression

Do not judge price alone.
Always consider what the buyer receives.

---
## Keyword Consistency (5 Points)

Evaluate consistency of keywords across:
* Title
* Description
* Tags
* Seller Profile

The same service should be communicated consistently throughout the gig.

---
# Overall Keyword Strength (0–100)
Keyword Strength measures how effectively the gig uses keywords for discoverability.
Evaluate:
* Primary keyword relevance
* Secondary keyword relevance
* Natural keyword placement
* Keyword consistency
* Search intent alignment
* Keyword coverage

Do not reward excessive repetition.
Natural usage is always preferred over keyword stuffing.

---
# Buyer Trust (0–100)

Buyer Trust measures marketplace credibility.
This score is **independent of SEO Strength**.
Use only:
* Average Rating
* Total Reviews
* Rating Distribution

Rules:
* No rating available → Buyer Trust = 0
* Higher average rating increases trust.
* More reviews increase confidence only when ratings remain consistently high.
* A poor rating should significantly reduce Buyer Trust.
* Mixed ratings should produce moderate trust scores.

Buyer Trust must never directly change the SEO Strength score.

---
# Package Review Score (0–100)
Evaluate:
* Value for money
* Package clarity
* Feature completeness
* Package differentiation

Provide:
* Numerical score
* Evidence-based explanation

---
# Pricing Review Score (0–100)
Evaluate:
* Price competitiveness
* Fairness relative to included features
* Logical progression between packages

Do not recommend arbitrary prices.
Recommendations must be justified using the provided package information.

---
# Score Consistency Rules
Before returning the final result, verify:
* A gig cannot receive a very high SEO Strength if multiple major components are weak.
* A gig cannot receive a very low SEO Strength if nearly every component is strong.
* Buyer Trust must never alter SEO Strength.
* Every recommendation should be consistent with the assigned scores.
* Every deduction must be explainable from the provided gig information.
