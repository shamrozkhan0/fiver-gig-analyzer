# GigBro Profile Guidelines
## Purpose
The seller profile helps buyers understand who the seller is, what they specialize in, and whether they can be trusted to deliver the advertised service.
Evaluate the seller profile using only the provided information:

* Profile Description
* Expertise
* About Profile

The goal is to improve professionalism, credibility, consistency, and relevance without inventing qualifications or experience.

---
# Primary Objective
The seller profile should:

* Clearly explain the seller's expertise.
* Support the services offered in the gig.
* Build buyer confidence.
* Demonstrate professionalism.
* Reinforce the seller's niche.

A buyer should understand who the seller is and why they are qualified.

---
# Profile Description
Evaluate whether the profile description:
* Clearly introduces the seller.
* Explains their specialization.
* Matches the services offered.
* Uses professional language.
* Is easy to read.
* Naturally includes relevant keywords.

The profile should strengthen the gig, not confuse it.

---
# Expertise Consistency
The provided expertise represents the seller's actual skills.
Verify that:
* The gig matches the expertise.
* The profile description matches the expertise.
* Packages are consistent with the expertise.
* Tags support the expertise.

If major inconsistencies exist, explain them.
Never recommend services outside the seller's expertise.

---
# About Profile
Use the "About Profile" information only as supporting context.
Examples include:
* Country
* Languages
* Response time
* Last delivery
* Other profile information

This information may contribute to buyer confidence but should not influence SEO Strength.

Do not penalize sellers for factors they cannot control, such as location or native language.

---

# Professionalism
Evaluate whether the seller presents themselves professionally.
Consider:
* Clear communication
* Professional tone
* Confidence without exaggeration
* Accurate representation of services

Avoid recommending exaggerated marketing claims.

---
# Keyword Relevance
The profile description should naturally reinforce the seller's specialization.
Relevant keywords should:
* Match the gig.
* Match the expertise.
* Be used naturally.

Never force keywords into the profile.
Readability is always more important than keyword frequency.

---
# Consistency
Verify consistency between:
* Profile Description
* Expertise
* Gig Title
* Gig Description
* Tags
* Packages

All components should represent the same primary service.
If inconsistencies exist, clearly explain them.

---
# Buyer Confidence
Evaluate whether the profile helps a buyer feel confident.
A strong profile should:
* Explain what the seller specializes in.
* Demonstrate focus.
* Avoid vague statements.
* Be honest and realistic.

Do not fabricate achievements, certifications, years of experience, or client history.

---
# Preserve Existing Profile
When optimizing:
* Keep accurate information.
* Improve wording and readability.
* Preserve the seller's writing style whenever practical.
* Remove unnecessary repetition.
* Improve keyword relevance naturally.

Optimization is preferred over complete replacement.

---
# When to Rewrite Completely
Generate a completely new profile description only if the existing one is:
* Empty
* Extremely short
* Unprofessional
* Unrelated to the seller's expertise
* Poorly written
* Impossible to improve effectively

Otherwise, edit and optimize the existing profile description.

---
# Final Validation
Before returning the optimized profile description, verify that it:
* Accurately represents the seller.
* Matches the seller's expertise.
* Supports the gig.
* Uses relevant keywords naturally.
* Builds buyer confidence.
* Is grammatically correct.
* Maintains a professional tone.
* Preserves the seller's identity whenever possible.
