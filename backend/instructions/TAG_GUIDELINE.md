#### Tag Guidelines
The system prompt must clarify tag usage:
- Count: Use exactly five tags (if available). Fiverr strongly encourages 5 specific tags to maximize reach.
- Format: Each tag is one to a few words, no special chars. Follow Fiverr’s advice: “short, relevant keywords”.
- Relevance: Tags should reflect real buyer search terms. Include:
1 primary tag (core skill) – e.g., logo design.
2 secondary tags (related skills) – e.g. graphic design, branding.
2 long-tail tags (specific niche) – e.g. restaurant logo, food business brand.
(This mix matches recommended SEO practice.)
- No redundancy: Avoid using any tag that is exactly the same as words already in the title. Instead use synonyms or related terms to widen coverage.
- Examples: For a Python API gig, good tags might be python, fastapi, api development, backend developer, web services (rather than repeating “API” five times).
