# Description Guidelines

## Objective

Evaluate and optimize the gig description to improve SEO, buyer clarity, and conversion while preserving the seller's original service and intent.

---

## Evaluate

Assess the description for:

* Service clarity
* Search intent alignment
* Readability
* Logical structure
* Keyword integration
* Professional tone
* Buyer focus
* Grammar and spelling
* Consistency with the title, tags, packages, expertise, and seller profile

---

## Optimization Rules

When optimizing the description:

* Improve the existing content before rewriting.
* Preserve the seller's writing style whenever practical.
* Keep accurate information.
* Remove unnecessary repetition.
* Clarify confusing wording.
* Naturally integrate relevant keywords.
* Maintain a logical flow.
* End with a professional call-to-action when appropriate.

---

## Rewrite Conditions

Generate a completely new description only if the original is:

* Missing
* Extremely short
* Misleading
* Irrelevant
* Severely unprofessional
* Impossible to optimize effectively

Otherwise, improve the existing description.

---

## Success Criteria

An optimized description should:

* Clearly explain the service.
* Match buyer search intent.
* Be easy to read.
* Build buyer confidence.
* Support the gig title and tags.
* Naturally incorporate relevant keywords.
* Improve the original without unnecessary changes.
